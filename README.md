## 📖 项目简介

**Videotomp3** 是专门为手机用户打造的音频提取利器。与市面上动辄几十 MB 且充满广告的转码软件不同，本项目追求的是**极致的纯净**与**原生速度**。

它直接调用 Android 系统底层的 `MediaCodec` 框架，通过物理剥离技术将视频中的音频轨道直接“抽离”出来。对于 M4A 格式，它实现了**真正的无损提取**，不经过任何重编码，瞬间即可完成任务。

### ✨ 核心功能
* **极速转换**：支持一键将视频转换为 **MP3** (高兼容性) 或 **M4A** (原生无损)。
* **毫秒级响应**：对于常规短视频，提取过程几乎在点击瞬间即可完成。
* **分区存储适配**：自动在手机 `Music/VideoToAudio` 文件夹下归档，方便查找。
* **完全开源**：没有任何后台行为，不申请多余权限，代码全透明。

---

## 🛠️ 技术实现
本项目采用 Android 原生 API 开发，未使用任何第三方臃肿库：
* **MediaExtractor**：负责精准定位视频流中的音频轨道。
* **MediaMuxer**：负责将音频数据重新封装为独立文件。
* **MediaStore API**：完美适配 Android 11+ 的分区存储限制。
* **Html.fromHtml**：在 UI 层面实现灵动的多色版权显示。
